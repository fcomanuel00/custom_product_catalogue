===========================
Product Catalogue Generator
===========================

Generate professional product catalogue PDFs from your Odoo database in seconds.

Installation
============

1. Download the module from the Odoo Apps Store.
2. Extract the archive into your ``addons`` folder.
3. Restart the Odoo server.
4. Go to *Apps*, click *Update Apps List*, and search for
   ``Product Catalogue Generator``.
5. Click **Install**.

Optional dependency: if the module *Website eCommerce* (``website_sale``) is
installed, the wizard will show an extra filter to only include products
published on the web. If not installed, the filter is hidden automatically and
everything else works exactly the same.

Configuration
=============

No configuration is required. Two user groups are created on install:

- **User**: can browse and download the catalogues generated by others.
- **Manager**: can generate new catalogues and send them by email.

Assign these groups from *Settings → Users & Companies → Users*.

Usage
=====

Generate a new catalogue
------------------------

1. Open the top-level menu **Catalogues**.
2. Click **New Catalogue**.
3. Choose:

   - Which products to include (all, a specific list, or by category).
   - Which pricelist to apply.
   - Which of the 4 styles (table, full card, category grid, configurable grid).
   - Image size and which fields to show.
   - Optionally, a custom HTML header, footer, or cover/back-cover PDFs.

4. Click **Generate catalogue**.

The PDF is generated immediately and a record is created under
*Catalogues → Catalogues generated* with all the parameters and the PDF
attached.

Regenerate an existing catalogue
--------------------------------

Open any existing catalogue, change the style or the fields shown, and click
**Regenerate PDF**.

Email a catalogue to a customer
-------------------------------

From a generated catalogue, click **Send by email**. The PDF is automatically
attached to the email composer. A ready-to-use template in Spanish is
included.

Styles
======

- **Style 1 — Compact table.** Small thumbnail + reference + description +
  price, one row per product. Ideal for dense catalogues.
- **Style 2 — Full product card.** Large image, big price, rich description.
  Variants appear in a mini-table inside the card.
- **Style 3 — Grid grouped by category.** Three-column layout with category
  headers automatically inserted.
- **Style 4 — Configurable grid.** Lets you pick 2, 3 or 4 columns per row
  depending on the level of detail you want.

Variant support
===============

For products with variants (colours, sizes, etc.), the parent product is shown
as a section and each variant is listed underneath with its attributes, image
and price. The image falls back to the parent's image if the variant does not
have its own.

Technical notes
===============

- Models: ``product.catalogue`` (permanent records), ``product.catalogue.wizard``
  (transient).
- Report engine: QWeb-PDF.
- PDF merging (cover / back-cover): ``pypdf`` (bundled with Odoo 19).
